# Zaalima-development-Data-analytics-project-2-AlphaPulse-Financial-Analytics

## 🚀 Project Overview

AlphaPulse Financial Analytics is an end-to-end data analytics project focused on analyzing stock market data to evaluate portfolio performance and risk. The project covers the complete pipeline from data collection to executive-level visualization.

---

## 🎯 Project Objective

* Analyze a diversified portfolio of stocks across multiple sectors
* Calculate key financial risk metrics
* Simulate future portfolio performance
* Build an interactive dashboard for decision-making

---

## 🛠 Tools & Technologies

* **Python** (Pandas, NumPy, yFinance)
* **Tableau Public**
* **GitHub**

---

# 📅 Project Breakdown

---

## 🔹 Week 1: Data Acquisition & Cleaning

* Collected historical stock data using yFinance
* Selected multiple stocks and S&P 500 index
* Handled missing values using forward fill method
* Cleaned dataset for further analysis

📂 Output:

* `raw_stock_data.csv`
* `cleaned_stock_data.csv`

---

## 🔹 Week 2: Quantitative Analysis

* Calculated daily and log returns using NumPy
* Built Monte Carlo Simulation (10,000 runs)
* Forecasted portfolio performance for 1 year
* Analyzed distribution of simulated results

📂 Output:

* `log_returns.csv`
* `monte_carlo_simulation.csv`

---

## 🔹 Week 3: Visual Storytelling (Tableau)

* Connected processed data to Tableau
* Built interactive dashboard
* Created charts:

  * Heatmap (correlation)
  * Stock trend analysis
* Enabled user interaction for better insights

📂 Output:

* Tableau Dashboard

---

## 🔹 Week 4: Finalization

* Automated stock data refresh using Python
* Calculated key financial metrics:

  * Annual Return
  * Volatility
  * Value at Risk (VaR 95%)
  * Maximum Drawdown
* Built Executive Summary dashboard for stakeholders
* Validated all metrics using Python

📂 Output:

* `portfolio_metrics_final.csv`
* Executive Dashboard

---

# 📊 Key Insights

* Identified portfolio risk using VaR and drawdown analysis
* Evaluated volatility and return trade-offs
* Simulated future performance using Monte Carlo methods
* Delivered insights through a clean executive dashboard
